# Tasks Management version 2
> **Demo: http://tasksmanagement2.herokuapp.com/**

## Libraries & Technique Used:
1. Bootstrap 3
2. React JS
3. React Hooks
4. Hooks Context ( for replacing redux )
5. TypesChecking

## Usage:
```
git clone https://github.com/18520339/Tasks-Management.git
cd Tasks-Management
npm install
npm start
```
